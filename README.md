# 🎬 Anime Clip Downloader

Automatically downloads the highest-viewed 4K twixtor anime clips from YouTube.

## Requirements
- Python 3.8 or higher → python.org/downloads
- yt-dlp → install by running: pip install yt-dlp

## Setup

1. Install Python from python.org/downloads
   - ⚠️ Tick "Add Python to PATH" during install

2. Install yt-dlp — open Command Prompt and run:
   pip install yt-dlp

3. Run the script:
   python downloader.py

## How to Run
Double-click START.bat — it installs everything and runs automatically.

Or in Command Prompt:
   python downloader.py

Clips are saved to the downloads/ folder automatically.
